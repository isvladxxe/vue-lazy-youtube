# vue-lazy-youtube

### Install:
``` bash
$ npm install vue-lazy-youtube -S
```
### Usage:
```js
import LazyYoutube from 'vue-lazy-youtube' 

export default {
    //...
    components:{ LazyYoutube }
    //...
}
```

```html
<template>
  <LazyYoutube
    vid="4JS70KB9GS0"
    alt="Как вставить двадцать видео с Ютуба и не скачать слона"
  />
</template>
```
